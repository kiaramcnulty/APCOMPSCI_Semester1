Primitive Data Types

After learning about variable initialization and assignment, you should be aware that data types are serious business. They can determine the success or failure of your project. Therefore, you should know them extremely well. This document should serve as a quick reference guide for the data types we will be using most often in this class. Research each of the terms below and write their definitions in the boxes below

 

int : 32-bit integer. An int only accepts whole numbers - no decimals, fractions, etc.

Double: larger float type that holds both bigger and more precise numbers. Basically accepts all real numbers, including decimals.

boolean: accepts value of true or false, typically determines whether or not a block of code will run.

float: a variable with a fractional value. Basically opposite of an integer.

char: typically means a letter of the alphabet A-Z (upper or lower case), the digits 0 to 9, and the underscore. A single piece of a string, int, etc.

short: at least 16 bits in length. Type of integer.

long: at least 32 bits in length. Type of integer.
